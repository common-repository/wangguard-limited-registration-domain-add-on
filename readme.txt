WangGuard Limited Registration Domain Add-On is an Add-on for WangGuard plugin. You can allow or disallow a list of domains.


== Description ==

With this WangGuard Add-on, you can allow or disallow domains for signup on your website. It is compatible with WordPress, BuddyPress and WooCommerce. WordPress Multisite has its own allow/disallow, so its not implementet.

[WangGuard plugin](http://wordpress.org/extend/plugins/wangguard) version 1.6+ is needed

= Why use WangGuard Registration Nocice Add-on? =

If you are a website that you want to allow to signup only cetains domain, with this add_on you can do it. Or maybe, you want to block certain domain that are not in WangGuard Domain list.


== Installation ==

1. Download and install WangGuard Registration Nocice Add-on

2. Activate plugins through the "Plugins" menu in WordPress (if is a Multisite, WangGuard needs to be activated for the Network)

3. Updates are automatic. Click on "Upgrade Automatically" if prompted from the admin menu. If you ever have to manually upgrade, simply deactivate, uninstall, and repeat the installation steps with the new version.
